# Command
1. install brew: '/usr/bin/ruby -e "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install)"'
2. update brew: brew update
3. install iterm2: brew cask install iterm2
4. install bash: brew install bash
5. install git: brew install git
6. install node: brew install node
7. install spectacle: brew cask install spectacle
8. install chrome
9. install vscode: brew cask install visual-studio-code
10. install unikey
11. Show only app running on Dock: defaults write com.apple.dock static-only -bool true; killall Dock
    Make default Dock: defaults write com.apple.dock static-only -bool false; killall Dock
